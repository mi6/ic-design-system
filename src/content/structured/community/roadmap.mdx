## Introduction

The purpose of sharing the roadmap is to give our community an insight into our strategic plan, goals and milestones.

Priorities may change because of organisation strategies, or community requirements. We will aim to keep this page up to date.

We welcome contributions of code, issues raised and more. You can read more about [how to contribute](/community/contribute).

Check out our <ic-link target="_blank" href="https://github.com/orgs/mi6/projects/2/views/1" rel="noreferrer noopener nofollow">GitHub</ic-link> for a more detailed view of our day-to-day activities.

## Recently shipped

### Components

- Optional CSS theme for native tables.
- Addition of cursor position to [IcTextField](/components/inputs/text-field) key down event.
- An update to [IcToggleButton](/components/inputs/toggle-button) to reflect IcToggleButtonGroup disabled state.
- Fix to select all rows in [IcDataTable](/components/data-display/data-table).
- Improvements to [IcTextField](/components/inputs/text-field) validation aria-live behaviour, addition of `validationAriaLive` prop.
- Addition of ability to hide [IcSelect](/components/inputs/select) menu option labels.

For a full list of recent component changes made, please read the <ic-link target="_blank" href="https://github.com/mi6/ic-ui-kit/blob/main/packages/web-components/CHANGELOG.md" rel="noreferrer noopener">changelog</ic-link>.

### Guidance website

- Update component gallery images.
- Update logos on website footer.
- Fix tree item becoming unselected on Code and Accessibility pages.

## Current focus

- Introduce more code examples for all components.
