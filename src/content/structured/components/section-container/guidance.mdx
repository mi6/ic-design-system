## Introduction

The section container is used by multiple ICDS components to align content more easily.

The components currently using the section container are:

- [Footer](/components/footer)
- [Hero](/components/hero)
- [Page header](/components/page-header)
- [Top navigation](/components/top-nav)

## Usage

When creating a custom component or a web page, a section container can be used to align and group
related content to make the end result easier to understand and digest for the end user.
