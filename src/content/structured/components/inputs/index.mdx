## Introduction

Input components are elements that act like a bridge, allowing users to provide data to an application. They enhance the usability of forms and make applications intuitive.

## Components

The current input components are:

- [Button](/components/inputs/button)
- [Checkbox](/components/inputs/checkbox)
- [Date input](/components/inputs/date-input)
- [Date picker](/components/inputs/date-picker)
- [Radio](/components/inputs/radio)

- [Select](/components/inputs/select)
- [Switch](/components/inputs/switch)
- [Text field](/components/inputs/text-field)
